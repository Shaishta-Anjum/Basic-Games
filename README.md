# Basic-Games
1)ROCK PAPER SCISSOR!!
In this game of Rock Paper Scissor there'll be a total of 5 Rounds. The rules for winning this game is:
Rock wins against Scissor
Paper wins agains Rock
Scissor wins against Paper
